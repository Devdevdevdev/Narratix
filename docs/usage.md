# Usage

1. Start the backend.
2. Start the frontend.
3. Open `http://localhost:5173`.
4. Paste your DeepSeek API key, unless it is already configured in `backend/.env`.
5. Choose a platform, content type, length and one of the restored Narratix tones.
6. Enter a brief or a post to answer.
7. Click "Générer le contenu".

Generated content appears in the result panel and is saved in local browser history.

## Key Storage

If "Stocker la clé dans ce navigateur" is enabled, the key is stored in browser `localStorage`.

Disable that option if you want to paste the key manually for each session.

## Local History

The generation history is stored in browser `localStorage`. Use "Vider" to clear it.

## Costs

Narratix does not charge anything. DeepSeek usage is billed directly by DeepSeek on the API key owner's account.
